Code example

KATA from CODEWARS “What is a narcissistic number? A Narcissistic Number is a positive number which is the sum of its own digits, each raised to the power of the number of digits in a given base. In this Kata, we will restrict ourselves to decimal (base 10). For example, take 153 (3 digits), which is narcisstic: 1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 153

The Challenge: Your code must return true or false (not ‘true’ and ‘false’) depending upon whether the given number is a Narcissistic number in base 10. This may be True and False in your language, e.g. PHP. Error checking for text strings or other invalid inputs is not required, only valid positive non-zero integers will be passed into the function.

function narcissistic(value) { var numbers = String(value).split(''); var sum = 0; for (let i = 0; i < numbers.length; i++) { parseInt(sum += numbers[i] ** numbers.length); } if (sum === value) return true; else return false; }
